Metodo que implementa el algortimo K-NN para un item a que recibe como parametro. Devuelve un diccionario con los k items con mejor prediccion de valoracion. Para la prediccion se usa la similitud entre ítems y se pondera con el rating dado por el usuario que pide la recomendacion del item a y con el rating promedio del ítem candidato a formar parte de los k-NN.

Parámetros
aItem para el cual se calculan los K-NN
ratingValoracion del usuario que pide la recomendacion al ítem a
Devuelve
Estructura de datos que contiene, para un conjunto de items determinado, sus predicciones
